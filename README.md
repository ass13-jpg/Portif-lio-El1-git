# Portfólio Git - CIn-UFPE 📚

Repositório criado para a atividade "EI1 — Portfólio Git" da disciplina de Desenvolvimento de Software (CIn-UFPE).
O objetivo deste repositório é demonstrar o domínio prático do Git e do GitHub, incluindo commits semânticos, uso de branches, Pull Requests e resolução de conflitos.

## 🗂️ Índice e Estrutura do Repositório

* **`/projeto/hello-world`**: Contém um mini-projeto, um sistema básico de tarefas em JavaScript usado para simular o desenvolvimento.
* **`/exercicios/ex01-commits`**: Evidências de commits realizados.
* **`/exercicios/ex02-branches`**: Contém evidências e anotações sobre a criação de branches e Pull Requests.
* **`/exercicios/ex03-conflito`**: Arquivo final provando a resolução de um conflito de merge.
* **`REFLEXAO.md`**: Documento com a minha reflexão crítica sobre o aprendizado da ferramenta.

## 🛠️ Tecnologias Utilizadas
* Git e GitHub
* JavaScript (para simulação do mini-projeto)
